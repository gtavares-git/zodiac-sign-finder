# Zodiac Sign Finder

Aplicação web desenvolvida em PHP para descobrir o signo zodiacal com base na data de nascimento do usuário.

---

## Funcionalidades

* Inserção de data de nascimento
* Consulta de signo via arquivo XML
* Exibição do nome e descrição do signo

---

## Tecnologias utilizadas

* PHP
* XML
* HTML5
* CSS3
* Bootstrap

---

## Estrutura do projeto

```bash
zodiac-project/
│
├── assets/
│   ├── css/
│   ├── js/
│   └── imgs/
│
├── layouts/
│   └── header.php
│
├── index.php
├── show_zodiac_sign.php
└── signos.xml
```

---

## Como executar o projeto

1. Instale o XAMPP
2. Coloque a pasta do projeto em:
   C:\xampp\htdocs\
3. Inicie o Apache no XAMPP
4. Acesse no navegador:
   http://localhost/zodiac-project
